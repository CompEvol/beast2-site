The following programs are distributed as part of the BEAST package

BEAST
Bayesian Evolutionary Analysis Sampling Trees
BEAUti
A simple user interface for creating input files to run BEAST.
LogCombiner
A program to combine log and tree files from multiple runs of BEAST
TreeAnnotator
A program for summarizing the information in a sample of trees produced by BEAST
TreeLogAnalyser
A program for analyzing the tree sets produced by BEAST
[DensiTree]
A program for visualising the information in a sample of trees produced by BEAST

The following programs are distributed separately but can be used in conjunction with BEAST

Tracer
A program for analysing results from Bayesian MCMC programs such as BEAST & MrBayes
[FigTree]
A program for viewing trees including summary information produced by TreeAnnotator
