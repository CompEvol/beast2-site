BEAST v2.7.8 Documentation: feast.mapping.DensityMapper

Entry point for running a Beast task, for instance an MCMC or other probabilistic analysis, a simulation, etc.
Utility for mapping probability density surfaces.

Inputs:

realParam, steps, logScale, distribution, logger, acceptAfterDistribCalculation

 

realParam
type: beast.base.inference.parameter.RealParameter***
Real parameter to vary.
Optional input

 

steps
type: beast.base.inference.parameter.IntegerParameter***
Number of steps to take between parameter bounds. 1 means useinitial value.
Optional input

 

logScale
type: beast.base.inference.parameter.BooleanParameter***
Whether to use log scale when stepping. Default false.
Optional input

 

distribution
type: beast.base.inference.Distribution***
Distribution to map.
Optional input

 

logger
type: beast.base.inference.Logger***
Logger used to store output.
Optional input

 

acceptAfterDistribCalculation
type: java.lang.Boolean
If true, call state.acceptCalculationNodes() after distribution has been computed. Default true.
Optional input. Default: true
