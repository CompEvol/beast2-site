BEAST v2.5.0 Prerelease Documentation: beast.evolution.speciation.CalibrationPoint

Specification of a single calibration point of the calibrated Yule.

Inputs:

taxonset, distr, parentOf

 

taxonset
type: beast.evolution.alignment.TaxonSet
Set of taxa. The prior distribution is applied to their TMRCA.
Required input

 

distr
type: beast.math.distributions.ParametricDistribution
Prior distribution applied to time of clade MRCA
Required input

 

parentOf
type: java.lang.Boolean
Use time of clade parent. Default is false.
Optional input. Default: false
