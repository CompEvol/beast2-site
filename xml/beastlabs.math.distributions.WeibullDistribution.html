BEAST v2.7.8 Documentation: beastlabs.math.distributions.WeibullDistribution

BEASTObject that performs calculations based on the State.
A class that describes a parametric distribution, that is, a distribution that takes some parameters/valuables as inputs and can produce (cumulative) densities and inverse cumulative densities.
Weibull distribution. for x>0 f(x;shape,scale) = scale/shape(x/shape)^{scale-1}e^{-(x/shape)^scale}

Inputs:

shape, scale, meanOne, offset

 

shape
type: beast.base.inference.parameter.RealParameter
shape parameter, defaults to 1
Optional input

 

scale
type: beast.base.inference.parameter.RealParameter
scale parameter, defaults to 1 unless meanOne=true, then scale is set to 1/Gamma(1+shape) so mean of the distribution = 1
Optional input

 

meanOne
type: java.lang.Boolean
Fix mean to one, ignore scale parameter
Optional input. Default: false

 

offset
type: java.lang.Double
offset of origin (defaults to 0)
Optional input. Default: 0.0
