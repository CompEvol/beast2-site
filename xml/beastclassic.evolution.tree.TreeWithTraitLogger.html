BEAST v2.7.8 Documentation: beastclassic.evolution.tree.TreeWithTraitLogger

Logs tree annotated with metadata and/or rates

Logable: yes, this can be used in a log.

Inputs:

tree, metadata, combine

 

tree
type: beast.base.evolution.tree.Tree
tree to be logged
Required input

 

metadata
type: beast.base.core.BEASTObject***
meta data to be logged with the tree nodes.If it is a trait associated with a node, the metadata will be stored with the nodes.Otherwise, the metadata will be listed before the Newick tree.
Optional input

 

combine
type: java.lang.Boolean
put all parameters in a single field
Optional input. Default: false
