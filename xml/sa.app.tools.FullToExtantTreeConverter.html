BEAST v2.7.8 Documentation: sa.app.tools.FullToExtantTreeConverter

Entry point for running a Beast task, for instance an MCMC or other probabilistic analysis, a simulation, etc.
utility for creating a tree set with extant nodes from a set of ancestral node trees

Inputs:

output, trees, threshold

 

output
type: beastfx.app.util.TreeFile
file to store converted tree set. If not specified, _extant is aded to input file name
Optional input

 

trees
type: beastfx.app.util.TreeFile
tree set file to be converted
Optional input

 

threshold
type: java.lang.Double
the threshold for identifying extant taxa. Ignored if negative.
Optional input. Default: -1.0
