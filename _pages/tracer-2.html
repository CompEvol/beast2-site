Tracer is graphical tool for visualization and diagnostics of MCMC output. It can read output files from MrBayes and BEAST.

Downloading

You can download Tracer from here: https://github.com/beast-dev/tracer/releases/latest

Using Tracer

The Divergence Dating tutorial is available describing how to use Tracer to analyse the output from BEAST.

Citing Tracer

The recommended citation for this program is:

  • Rambaut A, Drummond AJ (2007) Tracer v1.4, Available from http://beast.bio.ed.ac.uk/Tracer

For more information and to download it, go to the Tracer web page.
